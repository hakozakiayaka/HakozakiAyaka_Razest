# ゲーム内容
#=================================================
 プレイヤー2人、鬼1人の鬼ごっこゲームです。
 鬼は、街の中にある3種類のオブジェクトに変化しながらプレイヤーを捕まえます。
 鬼が時間内にプレイヤー2人を捕まえられたら、鬼の勝利になり、
 プレイヤーが時間内に鬼から逃げ切れると、プレイヤーの勝利となります。
